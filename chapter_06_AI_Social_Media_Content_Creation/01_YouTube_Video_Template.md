# 01 — YouTube Video Template

A video script that hooks in 5 seconds, teaches one thing, and converts curiosity into a comment.
Pull your **Hook · Story · Offer** from [00](00_Hook_Story_Offer_Planning.md) before you start.

> Tone: senior colleague over chai. Slow down 10–15% from conversational pace. Land pauses
> *after* punchlines, not before.

---

## Structure at a glance

| Section | Time | Job |
|---------|------|-----|
| Title | — | Earn the click in the feed |
| Hook | 0:00–0:15 | Pattern interrupt, no intro yet |
| Intro | 0:15–0:45 | Credibility + problem + promise |
| Body | the bulk | Teach in clear beats, show proof |
| Outro | last 45–60s | Recap + comment CTA + subscribe |
| Thumbnail | — | The real click driver |

---

## Fill-in template

### TITLE — offer 2–3 options
Patterns: number-led • "Stop doing X" • "I did X, here's what happened" • pain-point-led.
```
Option A: ____________________________________________
Option B: ____________________________________________
Option C: ____________________________________________
```

### HOOK (0:00–0:15) — max ~50 words
Pattern interrupt in the first 5 seconds. One pause beat before the reveal. End on a bridge
("Let me show you."). No "Hey guys, welcome back."
```
SPOKEN:
___________________________________________________

VISUAL SHOTS (4–5 quick cuts):
- ________________________________
- ________________________________
- ________________________________
```

### INTRO (0:15–0:45)
Credibility line → name the problem → state the principle → preview the roadmap → "Let's go."
```
CREDIBILITY: I'm ______, [role/proof] ______________
PROBLEM:     ________________________________________
PRINCIPLE:   ________________________________________
ROADMAP:     Today: (1) ____ (2) ____ (3) ____
```

### BODY — break into beats
One idea per beat. Each beat = spoken script + a one-line visual/B-roll note. Show real code,
real screens, real numbers — don't just say it, demonstrate it.
```
BEAT 1 — ____________________________________________
  SCRIPT:  _________________________________________
  VISUAL:  _________________________________________

BEAT 2 — ____________________________________________
  SCRIPT:  _________________________________________
  VISUAL:  _________________________________________

BEAT 3 — ____________________________________________
  SCRIPT:  _________________________________________
  VISUAL:  _________________________________________
```

### OUTRO (last 45–60s)
Recap the framework in one breath → frictionless comment CTA → future-value subscribe ask →
warm sign-off → action close.
```
RECAP (one breath): _________________________________
COMMENT CTA: "Comment [ONE WORD] below if ____________"
SUBSCRIBE: "Subscribe — next week I'm covering ______"
ACTION CLOSE: "Go open [tool] right now and ___________"
```

### THUMBNAIL CONCEPT
```
Face on left ~40%, big bold text on right ~60%.
TEXT (3–5 words max): ________________________________
BADGE (FREE / AUDIT / BREAKING): ____________________
Avoid the shocked-face cliché. High contrast, readable at phone size.
```

---

## Pre-publish checklist
- [ ] Title has a curiosity gap or a clear promise (2–3 options tested)
- [ ] First 5 seconds are a pattern interrupt — no intro animation, no "welcome back"
- [ ] Intro states credibility + problem + roadmap inside 30 seconds
- [ ] Every body beat has a visual/B-roll note and at least one real demo
- [ ] Outro has a ONE-word comment CTA (frictionless) + a specific subscribe reason
- [ ] Thumbnail readable at thumbnail size, 3–5 words
- [ ] No banned phrases, no fabricated numbers
