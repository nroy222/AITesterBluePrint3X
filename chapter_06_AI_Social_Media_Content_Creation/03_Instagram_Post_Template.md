# 03 — Instagram Post Template (Single Image)

One image, one idea, one caption. The simplest format — which means the image and the first
caption line do all the work. Pull from [00](00_Hook_Story_Offer_Planning.md).

> A single post lives or dies on two things: does the image stop the scroll, and does the first
> caption line earn the "more" tap.

---

## The image

Pick one of three jobs for the image:

| Image type | Use when | Make it |
|------------|----------|---------|
| **Quote card** | You have one screenshot-worthy line | Big bold thesis line, high contrast, minimal decoration |
| **Single-stat hero** | You have one striking real number | Faded "before" → glowing "after", one number dominates |
| **Mini-diagram** | You're explaining a small concept | 3–5 labeled boxes, readable at 30% zoom |

```
IMAGE TYPE: _________________________________________
HEADLINE ON IMAGE (≤8 words): _______________________
SUPPORTING LINE (optional): _________________________
WATERMARK/HANDLE: bottom-right, small, muted
```

Style default (dark infographic): navy background + subtle grid, gold headers, red for problems,
green for solutions, monospace for numbers, bold sans for labels. Mobile-readable at 30%.

---

## The caption

```
LINE 1 — HOOK (this is the preview; make it stop the scroll):
___________________________________________________

(blank line)

BODY — 3–6 short lines, one idea. → arrows for any list:
___________________________________________________
___________________________________________________

SCREENSHOT LINE (the quotable truth):
___________________________________________________

CTA — one ask:
"Save this for your next sprint." / "Comment [KEYWORD] for ____"
___________________________________________________

HASHTAGS (5–10, mix niche + 1–2 broad):
___________________________________________________
```

### Caption rules
- First line shows in the feed truncated — it must work alone as a hook.
- Whitespace between beats. Walls of text die.
- One CTA. If part of a content set, reuse the same comment keyword across [Reel](02_Instagram_Reel_Template.md)/[Carousel](04_Instagram_Carousel_Template.md).
- No emojis (1–2 max, strategic).

---

## Pre-publish checklist
- [ ] Image stops the scroll and states the idea in 3 seconds
- [ ] Headline on image ≤ 8 words, readable at 30% zoom
- [ ] Caption first line works as a standalone hook
- [ ] Exactly one idea, one CTA
- [ ] Screenshot-worthy line present
- [ ] 5–10 relevant hashtags
- [ ] Handle/watermark on image, bottom-right
- [ ] No banned phrases, no fabricated numbers
