# 04 — Instagram Carousel Template

7–9 swipeable slides (4:5) that work as a **save-worthy reference card.** Carousels get saved and
shared more than any other IG format — design them to be kept, not just seen. Pull from
[00](00_Hook_Story_Offer_Planning.md).

> One idea per slide. If a slide has two ideas, it's two slides. The swipe is the pacing.

---

## Slide map

| Slide | Job |
|-------|-----|
| 1 | **HOOK** — the scroll-stopper (mirror your Reel/LinkedIn hook) |
| 2 | The problem / the stakes |
| 3…N-1 | Build the idea, ONE point per slide, → arrows when listing |
| (one) | A code micro-snippet or a real number, if technical |
| N (final) | Payoff thesis line + CTA with a comment keyword |

### ManyChat / auto-DM (the lead-magnet engine)
```
KEYWORD (1 word, NEW per topic): _____________________
WHEN THEY COMMENT IT, AUTO-DM DELIVERS:
  - [PDF / checklist / repo link]: __________________
  - [Article / video link]:        __________________
REPLY PROMPT (1 line to start a convo): ______________
```

## Design spec (consistent set, all slides)

```
- Ratio: 4:5 (1080×1350) per slide
- Background: deep navy #0a0a1a + subtle grid texture
- Colors: gold/amber #f59e0b headers · red #ef4444 problems/warnings ·
          green #22c55e solutions · white body
- Fonts: monospace (JetBrains Mono) for code/numbers · bold sans (Inter) for labels
- Flat vector, no 3D, readable at 30% zoom
- Handle/watermark: small, muted gray, bottom-right
- Tooling: Ideogram / DAL·E for text-heavy slides, or build in Figma/Canva for perfect text.
  If a generator garbles text after 2–3 tries, switch to Figma/Canva.
```

### Caption (carousel)
```
First line = restate the hook. Body = 2–4 lines that tease the payoff without giving it away
(make them swipe). End with the comment-keyword CTA. 5–10 hashtags.
```

---

## Pre-publish checklist
- [ ] Slide 1 hook stops the scroll on its own
- [ ] One idea per slide, 7–9 slides
- [ ] At least one proof slide (code snippet or real number)
- [ ] Final slide = payoff thesis + comment-keyword CTA
- [ ] NEW, unused 1-word ManyChat keyword + auto-DM copy written
- [ ] Consistent design across all slides, watermark present
- [ ] Readable at 30% zoom
- [ ] No banned phrases, no fabricated numbers
