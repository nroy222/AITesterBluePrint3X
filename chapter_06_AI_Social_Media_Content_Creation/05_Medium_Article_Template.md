# 05 — Medium Article Template

Long-form (1,500–3,500 words; sweet spot 2,500–3,200) that argues one thesis with paragraphs,
proof, and honesty. Medium rewards depth and rewards trust. Pull from
[00](00_Hook_Story_Offer_Planning.md).

> Write in paragraphs, not bullet lists. The only exception is numbered step-headers in a
> technical tutorial. Bullets are where arguments go to die.

---

## Structure template

```
1. # Title
2. **Bold subtitle** (the full thesis + numbers + the promise — works as a standalone teaser)
3. - - -
4. Personal / contrarian opening anchor (a moment, a story, the line that gets you in trouble)
5. 4–7 ## sections that build the argument in paragraphs + occasional code
6. ## The Honest Caveats   (MANDATORY)
7. ## The Bottom Line
8. - - -
9. *Italic CTA*
10. Tags line
```

---

## Fill-in template

### TITLE — 8–15 words, specific, often a number or transformation
Patterns: "Should You [X] in 2026?" • "The 5 Lies About [Y]" • "I [Did X]. Here's What I Found."
```
TITLE: ______________________________________________
```

### BOLD SUBTITLE (first paragraph) — 4–6 sentences
Contains the FULL thesis, 3–5 specific real numbers, and the concrete promise. This is what
shows in search and email digests, so it must stand alone.
```
___________________________________________________
___________________________________________________
```

### OPENING ANCHOR
A moment or a blunt line — never a generic stat. The thing that earns attention.
```

```

### BODY SECTIONS (4–7, "##" headers, punchy + specific)
Every statistic attributed to a source. Never fabricate — if unsure, omit. Code samples where
relevant: real patterns, inline `code` and fenced blocks.
```
## ________________________________________________
   <paragraphs + optional code/data>

## ________________________________________________
   <paragraphs + optional code/data>

## ________________________________________________
   <paragraphs + optional code/data>
```

### ## The Honest Caveats (MANDATORY)
3–5 honest limitations. Where possible, make the bias cut **against** your own interest — it's
the fastest way to earn trust.


### ## The Bottom Line
One tight closing section. The reader should be able to act on this paragraph alone.

### ITALIC CTA (very end)
```

```

---

## Formatting for the Medium paste
Clean Markdown: `#` = Title, `##` = sections, `**bold**`, `*italic*`, `[text](url)`, `- - -` for
dividers, `>` for blockquotes. Note: Medium may need the first bold paragraph set manually as the
**Subtitle**.

---

## Pre-publish checklist
- [ ] Title 8–15 words, specific (number or transformation)
- [ ] Bold subtitle contains full thesis + 3–5 real numbers + the promise, stands alone
- [ ] Opens with a moment/contrarian line, not a stat
- [ ] Paragraphs, not bullets (except tutorial step-headers)
- [ ] Every statistic has a source; nothing fabricated
- [ ] "## The Honest Caveats" present, with at least one self-critical point
- [ ] "## The Bottom Line" present
- [ ] Italic CTA at the very end + tags line
- [ ] No banned phrases
