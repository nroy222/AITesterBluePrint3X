# 02 — Instagram Reel Template

A 15–45 second vertical video (9:16) that stops the thumb, teaches one micro-idea, and drives a
comment-keyword DM. Pull your **Hook · Story · Offer** from [00](00_Hook_Story_Offer_Planning.md).

> A Reel is a hook with a payoff stapled on. If the first second doesn't stop them, nothing else
> matters. Front-load everything.

---

## Anatomy of a Reel

| Beat | Time | Job |
|------|------|-----|
| Visual hook | 0:00–0:01 | Motion / face / bold text — stop the thumb |
| Spoken hook | 0:01–0:03 | Say the promise or the pain |
| Payload | 0:03–0:30 | 2–4 fast beats, one idea |
| Payoff + CTA | last 3–5s | The thesis line + the ask |

---

## Fill-in template

### ON-SCREEN TEXT HOOK (0:00–0:01)
Big, bold, 3–6 words, high contrast. Visible even on mute (most people watch muted).
```
TEXT: _______________________________________________
```

### SPOKEN HOOK (0:01–0:03)
```
___________________________________________________
```

### PAYLOAD — 2–4 fast beats (one idea total)
Each beat: one spoken line + the on-screen text/visual. Keep cuts fast. Show the screen / code /
result, don't just talk over a face.
```
BEAT 1  SPOKEN: ____________________  ON-SCREEN: ______
BEAT 2  SPOKEN: ____________________  ON-SCREEN: ______
BEAT 3  SPOKEN: ____________________  ON-SCREEN: ______
BEAT 4  SPOKEN: ____________________  ON-SCREEN: ______
```

### PAYOFF + CTA (last 3–5s)
```
PAYOFF LINE (the screenshot-worthy thesis): __________
CTA: "Comment [KEYWORD] and I'll send you ____________"
LOOP TRICK (optional): end on a line that flows back into the first frame for replays.
```

### CAPTION
```
First line = a second hook (caption preview is tiny — make it count):
___________________________________________________

Body (2–4 short lines, → arrows for any list):
___________________________________________________

CTA: Comment [KEYWORD] for [the lead magnet].
HASHTAGS (5–10, niche > broad): ______________________
```

### PRODUCTION NOTES
```
- Format: 9:16, 1080×1920
- Captions/subtitles: ON (burned-in or auto) — assume muted viewing
- Length target: 15–30s (sweet spot for completion rate)
- Audio: trending or original; keep voice clear over music
- First frame doubles as the cover — make it readable
```

---

## Pre-publish checklist
- [ ] On-screen text hook readable on mute in frame 1
- [ ] Spoken hook lands inside 3 seconds
- [ ] Exactly one micro-idea, 2–4 beats
- [ ] Subtitles burned in / enabled
- [ ] Payoff line is quotable
- [ ] One comment-keyword CTA (matches the [Carousel](04_Instagram_Carousel_Template.md)/[Post](03_Instagram_Post_Template.md) keyword if part of a set)
- [ ] 9:16, subtitles on, cover frame readable
- [ ] No banned phrases, no fabricated numbers
